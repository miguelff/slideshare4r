Class Slideshare::GroupList
In: lib/slideshare/model.rb
Parent: Object

Modelles a list of groups

The following is the structure of the XML that represents a list of groups, which will be unmarshalled into an instance of this class. 

       <Groups>
       <Group>
       ...
       </Group>
       <Group>
       ...
       </Group>
       </Groups>

An instance of GroupList has the following properties:

items
A list of group instances
